AKT Practice

AKT Practice is a web-based learning platform designed to help GP trainees prepare for the Applied Knowledge Test (AKT). It combines a structured question bank with topic-linked revision guides, allowing users to test knowledge and immediately reinforce learning. The system uses a flexible backend architecture so each question can surface one or more relevant revision guides, enabling scalable content growth without duplication. The platform prioritises clarity, performance, and clinically relevant learning over rote memorisation.

My Role

I worked as the technical implementer and systems integrator. I designed and implemented the database architecture, built the question and revision-guide linking logic, handled Supabase schema design and relationships, and integrated the frontend with the backend. I focused on making the system scalable and flexible so questions can link to multiple revision guides without data duplication, while keeping performance stable.

Tech used
Frontend: HTML, CSS, JavaScript
Backend & database: Supabase (PostgreSQL, REST API)
Hosting: Vercel
Version control: GitHub
Tooling: VS Code, GitHub Copilot, AI-assisted development workflows

Barely Soldered Website & Necrowarp Subsite

Barely Soldered is an indie game studio website built to showcase the developer and its projects, with dedicated subsites for individual games such as Necrowarp. The site features a dark-fantasy inspired aesthetic, responsive layouts, media galleries, and integrated support/contact systems, all designed to be performant, accessible, and easy to maintain without a traditional backend.

The platform was structured to scale as new games are added, with each title receiving its own pages for information, media, and player support. Deployment and hosting are handled via a Git-based workflow with Netlify, enabling fast updates and simple content iteration.

My Role

This was my first real client project, where I was responsible for implementing and deploying the support and contact system across the main site and the Necrowarp subsite. I built multiple Netlify Forms with custom subject routing, ticket ID generation, spam protection, and confirmation pages, while ensuring the system worked correctly across desktop and mobile devices.

I also handled real-world debugging and deployment issues, including asset path resolution, mobile browser behavior, and DNS/SSL configuration, and iterated on the UI based on direct client feedback. This project gave me hands-on experience shipping a live production feature while working closely with a client and adapting to evolving requirements.

Tech used HTML, CSS, JavaScript, Netlify Forms, GitHub, Netlify

Chinese Language Tutor Platform

A full-stack web platform designed to support Chinese language learning through structured practice, assessments, and tutor oversight. The system allows students to complete quizzes and tests, track their progress, and build learning streaks, while tutors can monitor performance, manage students, and view detailed analytics through a dedicated dashboard. The platform focuses on clarity, usability, and real-time data to support effective teaching and learning.

My Role

Full-Stack Developer
Designed and implemented the Tutor Dashboard, including student management, analytics, and detailed progress views.
Built secure role-based access control for tutors and students.
Integrated frontend with backend APIs to fetch real-time student performance data.
Implemented critical features such as progress reset, student deletion, and activity tracking.
Debugged and optimized data flow between MongoDB, backend services, and the UI.
Ensured the application is production-ready with proper error handling and state management.

Tech used Frontend: HTML5, CSS3, Vanilla JavaScript; Backend: Node.js, Express.js; Database: MongoDB (Mongoose); Authentication & Security: JWT-based authentication, Role-based access control (Tutor / Student); Deployment & Tools: RESTful APIs, supabase, Git & GitHub.
